# IRCTC Clone

A web-based **IRCTC (Indian Railway Catering and Tourism Corporation) Clone** built using **ASP.NET Core MVC**. This project allows users to search trains, book tickets, manage profiles, and includes an admin panel to manage stations, trains, and bookings.

## Features

### User Side
- User registration and login
- Forgot password functionality
- View train schedules
- Book tickets and view booking history
- Profile management
- Aadhaar verification simulation
- Responsive design for mobile and desktop

### Admin Side
- Admin login
- Create, edit, and delete stations
- Add, edit, and delete trains
- Manage train schedules and seat availability
- Secure dashboard with role-based access

---

## Technologies Used

- **Frontend:** HTML, CSS, Bootstrap 5, JavaScript  
- **Backend:** ASP.NET Core MVC, C#  
- **Database:** SQL Server  
- **Authentication:** Cookie-based authentication with Claims  
- **Security:** SHA256 password hashing  
- **Tools & Libraries:** jQuery, Chart.js, SweetAlert2, Emoji-mart
